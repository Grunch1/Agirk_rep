Editing button description

Button 1:

Saves a text entry with an associated reference link in the References section of the JSON file.

Button 2:

Assigns a text entry to a category (e.g., Individuals, Animals, Locations, Dates) and saves it in the respective section.

Button 3:

Saves a text entry along with an explanation or definition in the Footnotes section.
